# Culvia Project Summary

This file keeps concise repository metadata for GitHub settings, README maintenance, release notes, and documentation cross-checks.

## Short Summary

Culvia is a local-first photo culling, scoring, review, and delivery workbench for photographers.

## Scope

Culvia helps review a folder of photos, compare local scoring signals, inspect optional OpenAI-compatible vision critique, make manual pick/hold/reject decisions, and export selected photos or CSV results.

The project is intended as a local review and selection layer before editing or delivery. It is not a full RAW editor.
