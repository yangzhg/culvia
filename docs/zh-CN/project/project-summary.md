# Culvia 项目摘要

这个文件保存简要仓库元信息，用于 GitHub 仓库设置、README 维护、发布说明和文档交叉检查。

## 简短介绍

Culvia 是一个面向摄影师的本地优先选片、评分、复核和交付工作台。

## 范围

Culvia 帮助用户复核照片目录、比较本地评分信号、查看可选的 OpenAI-compatible 图片评价、做人工入选/待定/淘汰判断，并导出选中照片或 CSV 结果。

它适合作为精修或交付前的本地复核和选片层，不是完整 RAW 编辑器。
