# Culvia Preview Assets

This folder contains lightweight images for the GitHub README, social cards, community posts, and documentation.

## Files

- `culvia-github-preview.jpg`: primary README preview.
- `culvia-review-flow.jpg`: review and decision-flow preview.
- `culvia-social-preview.jpg`: recommended GitHub social preview image.

## Privacy Notes

The preview images are composite project assets generated from practice-photo material. They do not include original photo files, EXIF metadata, source folder paths, API keys, SQLite files, exports, or runtime caches.

Do not add raw shoots, generated app thumbnails, uploads, exports, local logs, or personal paths to this folder.
